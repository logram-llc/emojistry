## Quickstart

**Using git**:

```
git clone https://github.com/logram-llc/emojistry
cd emojistry
npm install
npm run build
npm run start
```

**Using Docker**:

```
docker run -p 8000:5173 ghcr.io/logram-llc/emojistry:latest
```

## Testing

```
npm run test
```
